# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute Costs**: $0.10 per hour per user
  - Average usage: 10 hours/month
  - Monthly compute cost per user: $0.10 * 10 = **$1.00**
  
- **Storage Costs**: $0.02 per GB per month
  - Average storage per user: 5 GB
  - Monthly storage cost per user: $0.02 * 5 = **$0.10**

- **Bandwidth Costs**: $0.01 per GB
  - Average bandwidth usage: 20 GB/month
  - Monthly bandwidth cost per user: $0.01 * 20 = **$0.20**

- **Total Cost per Active User**:  
  **$1.00 (Compute) + $0.10 (Storage) + $0.20 (Bandwidth) = $1.30/month**

### Pricing Tiers
1. **Basic Tier**: $5/month
   - Features: Local search, Basic analytics, Community support

2. **Pro Tier**: $15/month
   - Features: Local search, Advanced analytics, Priority support, Integration with coding tools

3. **Enterprise Tier**: $30/month
   - Features: Local search, Full analytics suite, Dedicated support, Custom integrations, Team collaboration tools

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$20 - $50** per user
  - Marketing campaigns, outreach, and sales efforts.

### Lifetime Value (LTV) Estimate
- Average user lifespan: 24 months
- Average revenue per user (ARPU): 
  - Basic Tier: $5 * 24 = $120
  - Pro Tier: $15 * 24 = $360
  - Enterprise Tier: $30 * 24 = $720
- LTV Range: **$120 - $720**

### Break-even Users Count
- Monthly fixed costs (marketing, salaries, infrastructure): **$5,000**
- Contribution margin per user:
  - Basic Tier: $5 - $1.30 = $3.70
  - Pro Tier: $15 - $1.30 = $13.70
  - Enterprise Tier: $30 - $1.30 = $28.70

- Break-even users count:
  - Basic Tier: $5,000 / $3.70 ≈ **1,351 users**
  - Pro Tier: $5,000 / $13.70 ≈ **366 users**
  - Enterprise Tier: $5,000 / $28.70 ≈ **175 users**

### Path to $10K MRR
- Target: $10,000 MRR
- Pricing and user distribution scenarios:
  1. **Basic Tier**: $5/month
     - Users needed: $10,000 / $5 = **2,000 users**
  
  2. **Pro Tier**: $15/month
     - Users needed: $10,000 / $15 ≈ **667 users**

  3. **Enterprise Tier**: $30/month
     - Users needed: $10,000 / $30 ≈ **334 users**

- **Optimal Path**: 
  - Focus on **Pro Tier** with **667 users** for a balanced approach to revenue and user engagement.
